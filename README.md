## 📌 Sobre o Projeto

Este repositório contém duas entregas principais:

### 1. Dashboard de Vendas Porsche
Um painel visual interativo totalmente planejado, estruturado e desenvolvido com o auxílio de **IA Generativa (ChatGPT Plus)**. O dashboard permite visualizar e analisar os principais KPIs e métricas de desempenho de vendas da Porsche.

### 2. Agente de IA para Consultoria em BI (Custom GPT)
Foi desenvolvido um assistente/agente especialista construído dentro da plataforma do ChatGPT Plus. Este agente atua como um **Consultor de BI e Desenvolvedor de Dashboards**, capaz de:
* **Analisar Perguntas de Negócio:** Entender objetivos e dores do usuário para sugerir quais métricas e visões agregam mais valor.
* **Avaliação e Tratamento de Dados:** Solicitar a base de dados do usuário, validar sua estrutura, identificar inconsistências e sugerir decisões/correções caso haja erros na base.
* **Recomendação e Criação:** Sugerir layouts, seleções de gráficos e construir ou direcionar a construção do dashboard ideal.
* **Interação Contínua:** Manter um diálogo ativo e consultivo com o usuário em todas as etapas do projeto.

## 💡 Como Funciona o Fluxo de Trabalho com o Agente de IA (Dashbo)

# Fluxo de Trabalho do Dashbo

O Dashbo segue um processo estruturado para transformar dados em dashboards executivos orientados à tomada de decisão.

1. **Recebimento dos Dados**

   * Recebe arquivos de dados (Excel, CSV e outros formatos suportados).

2. **Validação da Qualidade dos Dados**

   * Analisa a estrutura dos dados.
   * Identifica tabelas, colunas, tipos de dados, valores ausentes, duplicados e inconsistências.
   * Caso encontre problemas, gera um relatório e solicita a validação do usuário antes de prosseguir.

3. **Descoberta dos Dados**

   * Identifica automaticamente dimensões, métricas, datas, hierarquias e relacionamentos entre os dados.

4. **Entendimento do Negócio**

   * Solicita ao usuário as perguntas de negócio que o dashboard deverá responder.

5. **Definição dos KPIs**

   * Identifica os KPIs mais relevantes para responder às perguntas de negócio.
   * Sugere KPIs complementares e explica sua importância, cálculo e aplicação.

6. **Recomendação de Análises**

   * Sugere análises adicionais, como tendências, sazonalidade, comparações, correlações, segmentações, outliers e oportunidades de negócio.

7. **Planejamento do Dashboard**

   * Define o layout do dashboard.
   * Seleciona os gráficos mais adequados para cada KPI com base em boas práticas de visualização.
   * Apresenta o plano ao usuário e solicita aprovação.

8. **Geração do Dashboard**

   * Constrói um dashboard executivo no Canvas do GPT com layout inspirado no Microsoft Power BI.
   * Sempre que possível, utiliza os dados reais para calcular KPIs e preencher automaticamente gráficos e indicadores.

9. **Geração de Insights**

   * Produz um resumo executivo destacando tendências, riscos, oportunidades e recomendações de negócio baseadas nos dados analisados.

10. **Revisão de Qualidade**

    * Realiza uma validação final do dashboard verificando clareza, consistência visual, legibilidade e aderência às boas práticas de Business Intelligence.

11. **Evolução Contínua**

    * Após a entrega, oferece refinamentos, novos KPIs, dashboards analíticos, novas visualizações e personalizações de identidade visual quando solicitadas pelo usuário.
